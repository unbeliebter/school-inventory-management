2.) Inventarverwaltung
Das Inventar der Schule soll verwaltet werden. Dabei gibt es verschiedene Zustände in denen sich das Inventar befindet.

Projektiert, Angefordert, Bestellt, Geliefert, Eingesetzt, Ausgeliehen, Reparatur, Ausgemustert.

Es gibt zuständige Personen, die in verschiedenen Zuständen verantwortlich sind. Im Einsatz gibt es einen Lagerort. Bei Ausleihe ist kein Lagerort aber ein Verantwortlicher zu speichern.

Eine Oberfläche soll eine Übersicht aller Inventargegenstände liefern und Filtermöglichkeiten bieten. Zum Filtern hat ein Gegenstand mehrere Eigenschaften. Abteilung, Gruppe (EDV, Lehr-Lernmittel), Fach (z.B. Mathe, ..), Ort, Verantwortlicher. Auch der Zustand soll als Filter angeboten werden. Die Liste muss exportierbar (Zwischenablage oder CSV-Datei) sein.

Sie müssen eine Nutzerverwaltung konzipieren. Alle Lehrer können die Anzeige nutzen, Verantwortliche können Einträge machen. Administratoren können Stammdaten wie Abteilungen, Fächer, Gruppen, Lehrer, ... bearbeiten.
